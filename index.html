Nama Produk

Rekomendasi penamaan untuk 2 jenis absensi.

Agar customer cepat paham, gunakan nama yang langsung membedakan model pembayaran dan target pengguna.

Berlangganan

Presensi Cloud Pro

Nama ini cocok untuk paket Rp. 5.000/user/bulan karena terdengar modern, fleksibel, dan siap dipakai untuk perusahaan yang jumlah karyawannya bisa bertambah.

Rp. 5.000/ user / bulan
  • Cocok untuk perusahaan yang ingin layanan berjalan terus.
  • Lebih cocok untuk banyak karyawan, cabang, WFH, dan dinas luar.
  • Fitur dapat terus dikembangkan sesuai kebutuhan operasional.
Saran posisi: jadikan paket utama untuk bisnis, kantor, sekolah besar, toko bercabang, dan tim lapangan.
Sekali Bayar

Presensi Mandiri Lifetime

Nama ini cocok untuk paket Rp. 375.000 sekali bayar karena menekankan bahwa customer punya sistem absensi sederhana yang bisa dipakai jangka panjang.

Rp. 375.000sekali bayar
  • Cocok untuk instansi kecil yang ingin biaya ringan.
  • Fitur inti sudah cukup untuk absensi masuk, pulang, izin, cuti, dan rekap.
  • Tanpa biaya bulanan untuk penggunaan dasar.
Saran posisi: jadikan paket hemat untuk sekolah kecil, kantor kecil, komunitas, toko, atau instansi yang baru mulai digitalisasi.

Fitur Utama

Fitur utama yang paling dicari customer.

Bagian ini menjelaskan fungsi utama aplikasi dengan bahasa yang mudah dipahami, agar customer langsung tahu manfaatnya sebelum memilih paket.

Login Admin dan User

Admin mengatur sistem, sedangkan user cukup membuka HP untuk absen, melihat riwayat, dan mengajukan izin atau cuti.

Jam Kerja, Shift, dan Istirahat

Paket Cloud Pro mendukung jam masuk, jam pulang, multi shift, mulai istirahat, dan selesai istirahat. Paket Lifetime fokus pada jam masuk dan pulang utama.

Radius Lokasi Absensi

Absensi bisa dibatasi berdasarkan titik lokasi. Customer dapat memastikan user benar-benar absen dari area kerja yang ditentukan.

WFH dan Tag Lokasi Rumah

Khusus Cloud Pro, admin dapat mengatur WFH dan menandai lokasi rumah user agar absensi dari rumah tetap terkontrol.

Dinas Luar

Khusus Cloud Pro, user yang bekerja di luar kantor tetap bisa dipantau melalui lokasi tugas, tanggal, dan radius yang ditentukan admin.

Jurnal Harian

Khusus Cloud Pro, user dapat mencatat aktivitas harian sehingga pimpinan tidak hanya tahu siapa yang hadir, tetapi juga apa yang dikerjakan.

Rekap dan Ringkasan Modern

Admin dapat melihat ringkasan hadir, telat, izin, cuti, dan menarik rekap berdasarkan kategori tertentu untuk kebutuhan laporan.

Mutasi Cabang dan Waktu Kerja

Khusus Cloud Pro, jika user pindah cabang, berubah unit, atau berganti jadwal kerja, admin dapat menyesuaikannya dari dashboard.

Audit Log Aktivitas

Khusus Cloud Pro, aktivitas penting tercatat agar perubahan data lebih transparan dan mengurangi risiko saling menyalahkan.

1 User 1 HP

Khusus Cloud Pro, satu akun dapat dibatasi hanya untuk satu perangkat agar titip absen lebih sulit dilakukan.

Hari Raya dan Libur

Khusus Cloud Pro, admin dapat menambahkan hari libur agar aturan absensi mengikuti kalender kerja instansi.

Profil User dan Laporan Berlogo

Dashboard dapat menampilkan foto profil user, nama instansi, logo, serta download rekap dengan identitas perusahaan.

Pertanyaan Umum

Jawaban singkat sebelum customer menghubungi WhatsApp.

Bagian FAQ membantu mengurangi pertanyaan berulang dan membuat customer lebih siap untuk bertanya paket yang sesuai.

Apa bedanya paket bulanan dan sekali bayar?

Paket bulanan cocok untuk operasional yang lebih lengkap seperti multi shift, WFH, dinas luar, jurnal, audit log, dan rekap detail. Paket sekali bayar cocok untuk kebutuhan dasar dengan biaya ringan.

Apakah paket sekali bayar tetap ada admin dan user?

Ya. Paket sekali bayar tetap memiliki fitur login admin dan user, pengaturan jam, radius, izin/cuti, rekap, logo, dan nama instansi.

Apakah bisa pakai logo dan nama perusahaan sendiri?

Bisa. Kedua paket dapat menggunakan nama perusahaan atau instansi dan logo agar tampil lebih profesional.

Paket mana yang paling disarankan?

Untuk perusahaan yang butuh kontrol lengkap, pilih Presensi Cloud Pro. Untuk instansi kecil yang hanya butuh absen masuk, pulang, izin, cuti, radius, logo, dan rekap dasar, pilih Presensi Mandiri Lifetime.
